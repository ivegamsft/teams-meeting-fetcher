## Findings

### Critical Discovery
The Terraform deployment process does not include a pre-build step for Lambda code. This is a **deployment reliability blocker**:
- Any team member running `terraform apply` without first building the Lambda zip will immediately break the EventHub → Lambda → DynamoDB pipeline
- The Lambda function exists in AWS (created by Terraform), but contains placeholder code

### Technical Details
- Lambda source: `apps/aws-lambda-eventhub/` (Node.js/TypeScript)
- Build command: `npm install --production && Compress-Archive -Path * -DestinationPath lambda-deploy.zip`
- Deployment command: `aws lambda update-function-code --function-name tmf-eventhub-processor-dev --zip-file fileb://lambda-deploy.zip`
- EventHub consumer group alignment verified; no message loss observed after Lambda fix
