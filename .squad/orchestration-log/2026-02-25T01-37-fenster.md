## Fixes Applied

  - `iac/variables-combined.tf`
  - `iac/variables-local.tf`

### Lambda Code (apps/aws-lambda-eventhub/handler.js)
- Replaced `.subscribe().catch()` with try/catch wrapper
- `subscribe()` is synchronous; errors caught with try/catch, cleanup via `subscription.close()`
